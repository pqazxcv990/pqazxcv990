링크

strong 태그

  • 중요한 의미가 있는 텍스트
  • 텍스트 강조할 때 사용
  • 중첩도 가능함. 강조 효과는 동일하지만 구조적으로 중요한 부분임을 의미
  • b태그도 사용했었지만 지금은 잘 사용 안함.

이 시설은 관계자 외 출입금지입니다.

em 태그

  • 의미강조
  • 텍스트가 기울어짐
  • 중첩해서 사용가능 하지만 강조는 동일
  • i태그도 사용했었지만 지금은 잘 사용 안함.

여기서부터는 위험하니까 절대로 지나가지 마세요.

폼 구성하기

사용자와 상호작용해서 정보를 입력받고 서버로 전송하기 위한 양식

form태그

  • 폼 양식
  • action: 입력받은 값 전송할서버 url
  • method: 서버에 전송할때 송신 방식, 보안요구 post 아니면 get
  • action 서버 개발자가 사용

input 태그

  • 입력받는 요소
  • type 필수 name,value 선택

  • name: 입력요소 이름 서버로 전송될 때,이름으로 지정 식별함
  • value: 초깃값

label태그

  • 상호작용 요소에 이름 붙일 때 사용
  • 잘 사용하면 label태그 클릭만해도 상호작용 요소 선택가능
  • 웹 접근성 향상을 위해서라도 필수로 사용하길 권장
  • 암묵적 방법: 상호작용 요소 포함
  • 명시적 방법: 라벨태그 for 속성과 상호작용요소의 id속성을 같은 값으로 설정
